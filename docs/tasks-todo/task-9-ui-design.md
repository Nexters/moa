# Task 9: UI 고도화 - 피그마 디자인 적용 (P1)

## 목표

P0에서 구현한 미니멀 UI를 피그마 디자인 시스템 기반으로 고도화한다.

## 스펙 참조

- [MVP 스펙](../mvp-spec.md)
- 피그마 디자인 파일 (링크 추가 예정)

## 범위

### 적용 대상 화면

| 화면          | P0 상태      | P1 목표                      |
| ------------- | ------------ | ---------------------------- |
| 메인 패널     | 기본 스타일  | RunCat 스타일 2컬럼 레이아웃 |
| 설정 패널     | 기본 폼      | 디자인 시스템 적용           |
| 온보딩        | 기본 스텝 UI | 애니메이션, 일러스트 적용    |
| 축하 화면     | 미구현       | Confetti + 축하 UI 구현      |
| 트레이 아이콘 | 기본 아이콘  | 커스텀 아이콘 (활성/비활성)  |

## 구현 내용

### 1. 디자인 시스템 구축

Tailwind CSS를 활용하여 `src/styles/theme.css` 단일 파일로 테마 관리:

```css
/* src/styles/theme.css */
@import 'tailwindcss';

@theme {
  /* Colors - 다크모드 고정 */
  --color-primary: #007aff;
  --color-success: #34c759;
  --color-warning: #ff9500;

  --color-text-primary: #ffffff;
  --color-text-secondary: #8e8e93;

  --color-bg-panel: #1c1c1e;
  --color-bg-input: #2c2c2e;
  --color-bg-hover: #3a3a3c;

  --color-border: #38383a;
  --color-divider: #38383a;

  /* Typography */
  --font-sans:
    'Pretendard', -apple-system, BlinkMacSystemFont, system-ui, sans-serif;

  /* Shadows */
  --shadow-panel: 0 4px 24px rgba(0, 0, 0, 0.4);

  /* Radius */
  --radius-lg: 12px;
  --radius-md: 8px;
}
```

### 2. 메인 패널 고도화

**목표 UI (RunCat 스타일)**:

```

│                                                     │
│  ┌─────────────────────────┬─────────────────────┐ │

│  │                         │      │   ⚙️    │   │ │
```

**구현 사항**:

- 2컬럼 레이아웃 (70% 정보 / 30% 메뉴)
- 섹션별 구분선
- 금액 증가 애니메이션 (부드러운 카운트업)
- 다크/라이트 모드 대응

### 3. 설정 패널 고도화

- 뒤로 버튼 트랜지션 애니메이션
- 입력 필드 포커스 스타일
- 저장 버튼 로딩/성공 피드백
- 슬라이드 전환 애니메이션 (메인 ↔ 설정)

### 4. 온보딩 고도화

- 스텝 진행 애니메이션
- 랜덤 뽑기 시 주사위 굴리기 효과
- 일러스트/이모지 캐릭터 추가
- 완료 시 환영 애니메이션

### 5. 축하 화면 (Task 8과 연계)

- Raycast 스타일 Confetti 파티클
- 금액 강조 타이포그래피
- 축하 메시지 애니메이션

### 6. 트레이 아이콘 디자인

```
icons/
├── tray-active.png       # 근무중 (컬러)
├── tray-active@2x.png    # Retina
├── tray-idle.png         # 비근무 (모노크롬)
└── tray-idle@2x.png      # Retina
```
